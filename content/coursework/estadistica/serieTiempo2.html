Cargando los datos de ejemplo

Ocuparemos la dataframe orange2 del paquete datana del libro Anlisis de datos con el programa estadstico R: una introduccin aplicada de Salas-Eljatib (2021).

> library(datana)
> data(orange2)
> df <- orange2

Esta dataframe contiene mediciones temporales de dimetro del fuste en rboles de Naranjo, esto es un estudio longitudinal, es decir, existen varias mediciones para cada unidad muestral. Ejemplos de este tipo son tpicos en estudios de crecimiento, por ejemplo, ver crecimiento de rboles de Nothofagus obliqua en Salas and Garca (2006).

Grfico simple de serie de tiempo

Para realizar este ejemplo, Ud debe bajar a su computador el siguiente archivo que contiene la funcin a emplear.

> source("timeSerPlot.R")
> timeSerPlot(df, y="dap", x="tiempo", obs.unit = "arbol")

Grfico de serie de tiempo segregado por un factor

> timeSerPlot(df, y="dap", x="tiempo", obs.unit = "arbol",
+             factor1="sitio")

Grfico de serie de tiempo segregado por dos factores

> timeSerPlot(df, y="dap", x="tiempo", obs.unit = "arbol",
+             factor1="sitio", factor2= "especie")

Tambien puede invertir la posicin de los factores, es decir, dejar el factor especie como panel principal, y al factor sitio con una leyenda diferente.

> timeSerPlot(df, y="dap", x="tiempo", obs.unit = "arbol",
+             factor2="sitio", factor1= "especie")

Grfico de serie de tiempo segregado por dos factores, personalizado

> timeSerPlot(df, y="dap", x="tiempo", obs.unit = "arbol",
+             factor1="sitio", factor2= "especie",
+             factor2.col = T, only.lines = T,
+             xlab = "Tiempo (dias desde germinacion)",
+             ylab = "Diametro del fuste (cm)")
